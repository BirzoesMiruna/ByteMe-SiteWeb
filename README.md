#  ByteMe | Recipe Aplication

##  Overview

**ByteMe** is a full-stack web application designed for food enthusiasts to discover and manage recipes from around the globe. The project focuses on providing a personalized experience through a complete authentication system and user-specific content management.

This platform demonstrates modern web development practices, featuring a seamless integration between an **Angular** frontend and a robust **Node.js** backend.

---

##  Features

*  Secure Authentication: Full User System (Sign Up / Login / Reset Password).
*  Global Discovery: Browse and filter recipes based on different countries and cultures.
*  Personal Cookbook: Authenticated users can save their favorite recipes to their profile.
*  Responsive Design: A user-friendly interface optimized for all screen sizes.
*  Dynamic Rendering: Real-time content updates based on user interaction.
*  RESTful Communication: Clean client-server integration via HTTP requests.

---

##  Technologies Used

### Frontend
* **Angular** (Framework)
* **TypeScript** (Logic)
* **HTML5 & CSS3** (Structure & Styling)

### Backend
* **Node.js** (Runtime environment)
* **Express.js** (Web framework)

---

##  Functionality & Architecture

The application follows a **Modular and Scalable Architecture**:
1.  **User Flow:** Users can register and manage their accounts securely.
2.  **Data Handling:** The backend processes authentication tokens and manages user-specific data storage.
3.  **Integration:** The frontend communicates with the Express API to fetch recipes and update user preferences dynamically.

---

##  Project Purpose

This project was developed to master **Full-Stack development** challenges, specifically focusing on:
* Implementing secure authentication flows.
* Managing complex state between frontend and backend.
* Structuring a scalable codebase for both client and server sides.
